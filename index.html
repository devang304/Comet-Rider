COMET-RIDER

Download .zip Download .tar.gz View on GitHub

OVERVIEW:

This is a native Android application to provide UT Dallas students with a detailed schedule and route of the campus shuttle, Comet Cruiser.

FEATURES:

  • Reminders
  • Route Map
  • Street View of bus stops

PURPOSE:

I developed this application while learning Android Development using Java. My idea was to learn Android dev while working on an application that I could use on a daily basis.
